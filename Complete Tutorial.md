## üü¶ 1. Introduction to Six Sigma Black Belt

## üü™ 4. DMAIC Framework (Advanced)

### **Define Phase**
- Project charter creation  
- Problem & goal definition  
- SIPOC + Stakeholder mapping  
- High-level VOC analysis  

### **Measure Phase**
- Selecting measurement systems  
- MSA/Gage R&R  
- Establishing baseline metrics  
- Data collection & validation  

### **Analyze Phase**
- Root cause identification  
- Hypothesis testing  
- Regression analysis  
- ANOVA/MANOVA  
- Fishbone & 5-Why analysis  

### **Improve Phase**
- Solution design  
- Piloting solutions  
- Optimization using DOE  
- Mistake-proofing (Poka-Yoke)  

### **Control Phase**
- Control plans  
- SPC charts  
- Process audits  
- Sustaining improvements
